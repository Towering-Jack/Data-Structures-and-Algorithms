# C 语言实现的基础算法与数据结构库  

## 本项目使用纯 C 语言实现了常见的算法与数据结构，涵盖图算法、字符串匹配、排序、查找、线性结构、树结构等内容，适用于学习或快速参考。

## 📁 项目结构

buildWheel/

├── G_CriticalPath.c      # 关键路径算法

├── G_Kruskal.c           # 克鲁斯卡尔最小生成树算法

├── G_LGraph.c            # 邻接表图结构实现

├── G_MGraph.c            # 邻接矩阵图结构实现

├── G_Prim.c              # 普里姆最小生成树算法

├── HashSearch.c          # 哈希查找算法

├── KMP_match.c           # KMP 字符串匹配主算法

├── KMP_next.c            # KMP Next 数组生成

├── L_ArrList.c           # 顺序表实现

├── L_SingleLinkList.c    # 单链表实现

├── L_StaticLinkList.c    # 静态链表实现

├── Q_LinkQueue.c         # 链式队列实现

├── Q_LoopQueue.c         # 循环队列实现

├── S_ArrStack.c          # 顺序栈实现

├── S_DoubleStack_ShareSpace.c  # 共享空间双栈

├── S_LinkStack.c         # 链式栈实现

├── Sort.c                # 排序算法合集

├── T_AVLTree.c           # AVL 平衡二叉树

├── T_BinSearchTree.c     # 二叉搜索树

├── T_BinTree.c           # 二叉树基础实现

├── T_DisjointSet.c       # 并查集

├── T_MaxHeap.c           # 最大堆实现
